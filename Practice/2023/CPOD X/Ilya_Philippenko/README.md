Если хочется запустить код, стоит добыть API ключ, положить его в переменную окружения TMDB_API_KEY для TMDB и исполнить следующее: 

~~~
pip install -r requirements.txt
python tmdb/get_data.py
python main.py
python add_music.py
~~~
