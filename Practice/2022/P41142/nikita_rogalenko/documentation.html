Title

Formula 1 Knowledge base

Version Info

In this version only movies and books are supported as 'related creation'

Description

Formula One knowledge base, which contains information about drivers, events, results and other entities related to subject area.

Also contains "related creations" data - information about cultural impact of this racing series
